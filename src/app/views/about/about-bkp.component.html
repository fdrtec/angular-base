Este projeto foi baseado nas habilidades técnicas que o Desenvolvedor FullStack Fabiano Dias Ramires adquiriu durante estudos e prática profissional a partir de 2017.

O ponto forte do portfólio é o código fonte da infra estrutura montada para demonstração de conhecimento. O portfólio se divide em: FrontEnd com Angular(TypeScript): GitHub FrontEnd. Backend com springBoot(Java):GitHub BackEnd. Banco de dados NoSQL(MongoDB)

O FrontEnd em Angular, foi estruturado com módulos para carregamento por demanda(lazyLoading = garantia de performance), módulos de rotas por contexto (componentes), Resource services abstratos , usando polimorfismo de modelos para reaproveitamento de código e escrito na nomeclatura REST para atingir qualquer api. Confecção de Componentes customizados e organização bibliotecas auxiliares a nível de compartilhamento (Shared). Construção respeitando o padrão MVC.
